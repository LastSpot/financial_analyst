# Portfolio Guide

This portfolio guide is based on **volatility regimes** measured by the **CBOE Volatility Index (VIX)**. Allocation weights are adjusted by regime to balance growth exposure with downside protection.

## Volatility Regimes

| Regime            | VIX range | Description        | Guide                    |
|-------------------|-----------|--------------------|--------------------------|
| Low volatility    | &lt; 15   | Complacent market  | [low_volatility.md](low_volatility.md)   |
| Normal volatility | 15–25     | Typical range      | [normal_volatility.md](normal_volatility.md) |
| High volatility   | 25–35     | Elevated fear      | [high_volatity.md](high_volatity.md)   |
| Panic volatility  | &gt; 35   | Stress / crisis    | [panic_volatility.md](panic_volatility.md)   |

## Asset Categories

- **QQQ** — Nasdaq-100 (broad tech/growth).
- **SMH** — Semiconductors sector.
- **High conviction** — Concentrated ideas.
- **Hedge** — Defensive / tail-risk exposure.

Use the linked guides for exact weight recommendations in each regime.
